arr与arr1均是空数组,长度为0,但是new Array需要调用Array的构造器,效率比前者低点

concat 连接两个或更多的数组，并返回结果

		var arr1=[1,2,3];
		var arr2=[4,5,6];
		var arr3=[7];
		var arr=arr1.concat(arr2,arr3);
		arr-result:1,2,3,4,5,6,7

join 把数组中的所有元素放入一个字符串。元素是通过指定的分隔符(默认",")进行分隔的。

		var arr=[1,2,3];
		var str=arr.join("-");
		str-result:1-2-3

pop 删除并返回数组的最后一个元素
